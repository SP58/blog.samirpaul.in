Online PDF Compression Tool

About The Project: An online PDF file compression tool to reduce the size of a .pdf file. Python Flask is used to upload the file to a temporary location on the server. In the backend, using the PDFNetPython library that file gets reduced and saved to its final location. After download, the files are automatically deleted from the server after 1 hour. Technologies used in this project: Python3, Flask, C, Shell, Nix, Replit, Git, HTML, CSS, JavaScript....

This webapp uses Huffman Coding for Text Compression and De-compression. Made with JavaScript, HTML5 and CSS3. This tool assigns a variable-length code to the characters of the uploaded file based on the frequency of occurrence.
